评价体系说明：综合稳定性评估方案 3.0

本评价体系综合考虑基金收益率趋势稳定性和偏离度异常检测，对短债基金进行综合稳定性评估。排序指标是综合稳定性指标，数值越小表示基金越稳定。

第一步：异常值检测
偏离度 = |基金收益率 - 同类基金均值| / |同类基金均值| 异常点判定：偏离度 > 0.5

对每个时间段的收益率计算与同类基金均值的偏离程度，当偏离度超过50%时标记为异常点，统计异常点总数。

第二步：趋势稳定性计算
变化率1 = |3个月收益率 - 1个月收益率| / |1个月收益率| 变化率2 = |6个月收益率 - 3个月收益率| / |3个月收益率| 变化率3 = |1年收益率 - 6个月收益率| / |6个月收益率| 变化率4 = |3年收益率 - 1年收益率| / |1年收益率| 基础稳定性 = (变化率1 + 变化率2 + 变化率3 + 变化率4) / 4

计算相邻期间收益率的变化率，评估基金收益趋势的平滑程度。变化率越小代表收益变化越平稳。

第三步：综合指标计算
异常惩罚 = 异常点数量 × 0.5 综合稳定性指标 = 基础稳定性 + 异常惩罚

将基础稳定性与异常惩罚相结合，得到最终的综合稳定性指标。该指标综合考虑了收益趋势稳定性和异常偏离情况。

评价规则
  1. 综合稳定性指标排序： 依据综合稳定性指标升序排序（数值越小越稳定）
  2. 基础稳定性： 衡量收益趋势的平滑程度，数值越小越好
  3. 异常惩罚机制： 每发现一个异常点增加0.5的惩罚值
  4. 稳定性标准： 综合稳定性指标 ≤ 2.0 为优秀稳定基金
指标解释
  • 基础稳定性： 反映收益变化的平稳程度，理想值应接近0
  • 异常点数量： 收益数据中偏离度超过50%的异常点数量
  • 综合稳定性指标： 综合考虑趋势稳定性和异常检测的最终评估指标
